API Overview

The Azure Local Releases API provides two main endpoints:

  • /api/releases - Access release information with advanced filtering by support status, release train, build type, and more
  • /api/releasetrains - Get release train metadata and status information

Both endpoints support pagination and flexible filtering options. For complete parameter documentation and to test the API interactively, visit the interactive documentation.
