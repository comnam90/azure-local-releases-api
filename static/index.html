Azure Local Releases API

REST API for Azure Local release information with advanced filtering and caching

GET /api/releases

Get Azure Local releases with optional filtering

Query Parameters:

  • supported - Filter by support status (true/false)
  • releaseTrain - Filter by release train (e.g., 2411, 2505)
  • buildType - Filter by build type (Feature, Cumulative)
  • limit - Limit number of results
  • offset - Offset for pagination
Examples:
/api/releases - All releases
/api/releases?supported=true&limit=10 - Latest 10 supported releases
/api/releases?releaseTrain=2505 - All releases from 2505 train

GET /api/releasetrains

Get Azure Local release trains information

Query Parameters:

  • supported - Filter by support status (true/false)
  • limit - Limit number of results
  • offset - Offset for pagination
Examples:
/api/releasetrains - All release trains
/api/releasetrains?supported=true - Supported release trains
